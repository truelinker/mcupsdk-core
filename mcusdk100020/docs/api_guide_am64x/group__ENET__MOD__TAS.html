  • inArgs: EnetTas_GenericInArgs
  • outArgs: Enet_Version

Type: Synchronous.

ENET_TAS_IOCTL_SET_ADMIN_LIST 

Set the admin list parameters of the TAS module.

IOCTL parameters:

  • inArgs: EnetTas_SetAdminListInArgs
  • outArgs: None

Type: Asynchronous.

ENET_TAS_IOCTL_GET_OPER_LIST_STATUS 

Get the status of the operational list update.

This IOCTL should be called after ENET_TAS_IOCTL_SET_ADMIN_LIST to verify admin list to operational list copy at specified time. This IOCTL should be called until ENET_TAS_OPER_LIST_UPDATED received as output.

IOCTL parameters:

  • inArgs: EnetTas_GenericInArgs
  • outArgs: EnetTas_OperStatus

Type: Synchronous.
